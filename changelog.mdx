## v1.0 — Initial release

_March 2026_

Recontrol launches on iOS and Android.

### Core features

**Accounts**
- Create financial accounts across four types: asset, liability, expense, and income
- Set opening balances at account creation
- View per-account balance history

**Transactions**
- Record double-entry transactions between any two accounts
- Set transaction date, amount, and optional description
- Back-date transactions to correct historical balances
- Edit and delete transactions with automatic balance correction

**Spaces**
- Personal space created automatically on signup
- Create shared spaces for households and groups
- Invite members by email address or phone number — invitations wait for recipients who have not yet signed up

**Access control**
- Two default roles per space: admin (full access) and member (full access to accounts and transactions; read-only elsewhere)
- Create custom roles with any combination of permissions across 5 resources and 4 actions
- Role changes take effect immediately for all current holders

**Insights**
- Balance history per account, updated on every transaction
- Aggregate queries: sum, count, average, min, max across any filtered set of accounts
- Date-range filtering

**Authentication**
- Email and password sign-in
- Sign in with Apple

**Account management**
- Account deletion from Settings — permanently and irreversibly deletes all data with no backups
- Privacy Policy and Terms of Service available in-app and at recontrol.money
